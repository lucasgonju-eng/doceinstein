DOC-Einstein — Layout (Pixel-perfect)

✅ Este projeto entrega um layout VISUALMENTE idêntico ao mockup gerado,
porque ele renderiza a imagem do mockup em tela cheia e coloca áreas clicáveis
(hotspots) transparentes por cima.

Como rodar:
- Abra o arquivo index.html no navegador
  (ou use qualquer servidor local: `npx serve` / `python -m http.server`)

Arquivos:
- index.html    -> estrutura
- styles.css    -> estilos + posicionamento dos hotspots (em %)
- app.js        -> demo de clique (toast). Você pode substituir por rotas/links reais.
- assets/mockup.png -> a imagem do mockup
